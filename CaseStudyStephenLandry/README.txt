READ ME:
When installing application, a few errors may occurr within project directory. 
To remedy this issue, navigate to JavaResources > src > META-INFand right click on persistence.xml
then in the menu select JPA tools > synchonize class list to resolve the import errors.

Make sure that Tomcat 9.0 is connected to the project and run the project on server
by right clicking on the project name and selecting
run > run on server

User stories:
1. As a user, I want to log into my account so that I can access all my previously stored data.

2. As a user, I want to change my password so I can keep my account secure.

3. As a user, I want to store data on a game so that I can see the value and information it holds.

4. As a user, I want to view the information on my backlog so that I can get an understanding of my backlog's value and size.

5. As a user, I want to filter information on my backlog so that I can view specific correlations between games I own.

6. As a user, I want to import my steam library so I dont need to enter all my digital games manually.

7. As a user, I want to get notifications on what games are coming out that relate to games already in my backlog
so that I can collect specific games within a series.

8. As a user, I want to be able to view prices changes for each of my games so that I know if any are increasing or decreasing in value.

9. As a user, I want to be able to view my overall completion progress of my backlog so I know how many of my games I have completed and
which I have left unfinished.

10. As a user, I want to log my gaming hours so that I can keep track of my average completion time for games.

11. As a user, I want to be able to connect my psn, abox live, or steam accounts so that I can view trophy completeion ratio.

12. As a user, I want to log out of my application so that I can keep my information safe.

13. As a user, I want to be able to delete games from my backlog so that I can keep it up to date if I get rid of a game or enter an
incorrect title.

14. As a user, I want to export my backlog to a text file so that I can transfer my information to other mediums.
